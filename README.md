SmartRestro – Restaurant Management System

SmartRestro is a web-based restaurant management system built using Laravel. It allows customers to browse food items, place orders, and book tables, while providing an admin panel to manage menu items, orders, and bookings efficiently.

The system is designed with a clean UI and practical workflow, making it suitable for small to medium restaurant operations.

🚀 Features
👤 User Side
User registration and login
Browse food menu
Add to cart and place orders
Table booking system
View order history
Profile management
🧑‍💻 Admin Side
Admin dashboard with analytics
Manage food items (Add/Edit/Delete)
Manage orders
Manage table bookings
Manage users
🛠️ Tech Stack
Frontend: HTML, CSS, Bootstrap / Tailwind
Backend: PHP (Laravel)
Database: MySQL
Server: XAMPP
📦 Key Modules
Authentication System
Order Management
Table Booking System
Admin Dashboard
Invoice Generation
🎯 Purpose

This project demonstrates a complete CRUD-based web application with authentication, admin control, and real-world restaurant workflow implementation.

⚠️ Note
Payment system is simulated (no real payment gateway)
Designed primarily for academic and learning purposes
